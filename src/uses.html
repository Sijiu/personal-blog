Wes Bos does this thing that I think is cool. He has a page on his site called Uses, where he lays out all of the tools and things that he uses. I think that's neat, helps developers come together, and can provide neat new tools to people that have never heard of them before. So, without further ado:

Editor/Terminal

  • VS Code! It's super awesome. Awesome enough to pull me over from Sublime Text even.
  • I use a theme called An Old Hope. Star Wars!
  • Currently, my favorite font is Fira Code. I'm a big fan of the programming symbol ligatures.
  • I just started using Hyper Term because it works on Mac and Windows, so I don't have to mentally adjust between home and work. Also, styling and customization is in JavaScript and CSS, which is easier for me. Also. HyperPower. Trust me.
  • Zsh and oh-my-zsh

Apps

  • Alfred to replace Mac Spotlight
  • Spectacle for Mac window management
  • Spark for email. It's so smart/intelligent!
  • Duet for using Ipad as another screen. Worth. It.
